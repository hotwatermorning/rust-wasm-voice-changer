## How to run

1. Install [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)
2. Build Rust code.

    ```sh
    ./wasm-audio/build-and-copy.sh
    ```

3. Start the server.

    ```sh
    yarn install
    yarn start
    ```
